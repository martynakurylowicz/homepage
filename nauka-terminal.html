Terminal

Czarne okienko

Ścieżka: Start + należy wpisać "cmd"

Do skakania po poleceniach można użyć strzałek góra i dół.

Polecenia:

  • cd 'lokalizacja' - przenosi do konretnego folderu
  • cd - wyświetlenie aktualnej ścieżki
  • dir - robi listę zawartości lokalizacji
  • mk dir 'nazwa' - tworzy nowy katalog w lokalizacji
  • code . - odpalenie programu
  • cd .. - wejdź do katalogu wyżej
  • git - wyszukanie informacji o aplikacji GIT
  • git config --global user.name "Imię i nazwisko" - nadanie oficjalnej nazwy użytkowanika
  • git config --global user.email "mail@mail.com" - nadanie oficjalnego maila użytkownika
  • git config --get user.name - sprawdzenie aktualnego użytkownika
  • git init - założenie nowego repozytorium / repozytorium = projekt
  • git status - informacje na temat danego projektu
  • git add . - dodaje do śledzenia wszystkie pliki w danym katalogu
  • git commit -m "opis zmian po angielsku" - tworzenie komita/ zapisanie aktualnej pracy
  • git push - eksportowanie zmian na githuba / do zrobienia po zapisaniu commita
  • gitk - historia zapisów
  • cls - czyści terminal
